# 画面設計（TyranoScript Web / スマホファースト）

## Global Styles

* レイアウト方針: 基本はフルスクリーン1カンバス（背景/立ち絵/メッセージ枠の重ね合わせ）。UIは絶対配置 + Flex（ボタン整列）。

* ブレークポイント:

  * Desktop（>=1024px）を基準。

  * Tablet/Phoneは、クイックメニューをアイコン化して折りたたみ、選択肢の縦積みを強制。

* デザイン・トークン（例）

  * 背景: #0b0f19（暗転/帯）

  * 文字色: #ffffff（本文） / #ffe8b0（強調）

  * アクセント: #ff5a7a（決定）

  * フォント: 日本語可読性優先（例: Noto Sans JP / システムフォント）

  * 本文サイズ: 24px（披露宴の遠目視認を優先）

  * 行間: 1.6

  * ボタン: 高さ44px以上、hoverで明度+10%、activeで縮小(0.98)

***

## 1) タイトル/エピソード選択

### Layout

* 画面全体を背景画像で覆い、中央にカード（幅720px）を配置。

* カード内はCSS Grid（2列）：左にタイトル/説明、右にエピソードリスト。

### Meta Information

* title: 「のぞみさメモリアル」

* description: 「披露宴向け5エピソード分岐ノベルゲーム」

* og:title / og:description / og:image: assets/ogp.png

### Page Structure

1. 背景（静止画 or ループ動画）
2. タイトルカード
3. フッター（著作権/注意書き）

### Sections & Components

* タイトルロゴ

  * ロゴ画像 + サブコピー（例: 「選択で変わる、ふたりの物語」）

* メインCTA

  * 「つづきから（直近セーブ）」

  * 「はじめから（エピソード選択へ）」

* エピソード一覧

  * 1〜5を縦リスト表示（各行に「タイトル」「所要時間目安」「既読/未読」）

  * ロック中は非活性（例: ep1未読ならep2以降ロック）

* 運営/設定（モーダル）

  * 音量（BGM/SE/Voice）スライダー

  * 文字速度・オート速度

  * スキップ（既読のみ/全て）

  * 「データ初期化」ボタン（確認ダイアログ必須）

* フルスクリーン誘導

  * 「F11 or ボタンで全画面」を明記

## 2) ゲーム本編（共通）

### Layout

* 背景→立ち絵→演出レイヤ→メッセージUI→オーバーレイ（メニュー/セーブ等）の順で重ねる。

* クイックメニューは画面右下に横並び（Flex）。

### Meta Information

* title: 「エピソード再生中｜のぞみさメモリアル」

* description: 「選択肢で分岐するノベルシーン」

* ogはタイトルと同一で可（静的サイトのため）

### Page Structure

1. シーン表示レイヤ（背景/立ち絵）
2. メッセージ領域（下部固定）
3. クイックメニュー
4. オーバーレイ（選択肢/バックログ/セーブ/設定）

### Sections & Components

* メッセージウィンドウ

  * 下部固定、高さ30〜35%程度

  * 名前枠（左上） + 本文（ルビ非必須）

  * クリック/Enterで次へ、Escでメニュー

* 選択肢UI（分岐）

  * モーダルまたはメッセージ上にカード表示

  * ボタンは縦積み（最大4つ）、各ボタンは短文 + 補足（任意）

  * 選択確定時にSE＋フェードで遷移

* クイックメニュー

  * Auto / Skip / Save / Load / Log / Title

  * 状態表示（Auto中ハイライト、Skip中ハイライト）

* バックログ

  * 直近N行（例: 200）をスクロール表示

  * 画面半透明（背景は暗転）

* セーブ/ロード

  * 20スロットのグリッド

  * 各スロットに「章名」「日時」「短いテキスト抜粋」

* 設定

  * タイトル画面と同等項目

### Interaction / Animation

* フェード（0.3〜0.6s）を基本。

* 場面転換はフェード＋BGMクロスフェード（任意）。
